# Odelle Nyse Project Brief

## Project Overview
Odelle Nyse is a CBT-focused personal assistant app built with Flutter and LiveKit, designed to help users self-actualize by understanding the spirit within them. The app works to elevate conscious awareness through the integration of cognitive behavioral therapy and spiritual practices.

## Core Requirements
1. Maintain and enhance the glassmorphic UI design
2. Implement placeholder components with functional, visually appealing alternatives
3. Use more saturated colors while maintaining the aesthetic
4. Add animations and interactions to improve user experience
5. Ensure proper LiveKit integration for voice assistant functionality

## Technical Stack
- Flutter
- LiveKit for voice/audio functionality
- Google Fonts for typography
- Glassmorphic UI design patterns

## Components to Implement
1. Home Screen:
   - Hero's Journey card
   - Recent Insights card (with radar graph)
   - Daily Affirmation card
   - Thought/Emotion/Behavior section

2. Voice Assistant Screen:
   - AudioWaveformVisualizer
   - TranscriptionView
   - VoiceControlBar
